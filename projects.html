Name:
Monroe County Safety Feed
Website:
http://mcsafetyfeed.org/
Source Code:
https://github.com/thequbit/mc911feedwatcher
Description:
Monroe County, NY provides a near-real-time non-violent 911 calls via an RSS feed. A set of scrips has been written and setup to pull the contents of this feed at a regular interval and save the information to a database. A website is avaialable to view these 911 calls.
Technologies Used:
Python, BeautifulSoup, PHP
Project Lead:
Tim Duffy
Name:
Traffairious
Website:
http://traffairious-facts.rhcloud.com/
Source Code:
https://github.com/hhroc/traffairious
Description:
Traffairious is a joint project between WXXI, Innovation Trail, and Hacks/Hackers Rochester. There have been a number of studies that link the pollutants produced by motor vehicles to health effects in children. The Traffairious project uses New York State Department of Transportation and New York State Department of Education to provide a visualization of what schools are within 1000 feet of a roadway that has over 50,000 cars daily. All schools are displayed on the map, and roads over 25,000 and over 50,000 are labeled.
Technologies Used:
Python, Flask, Mako, Leaflet.js, Bootstrap
Project Lead:
Remy DeCausemaker and Tim Duffy

Name:
Monroe County Jail Census Tools
Website:
Source Code:
https://github.com/thequbit/mcjailcensus
Description:
The Monroe County, NY jail system posts (most) days posts the current census. The list is posted in the form of a very large PDF document. This document has some odd formatting for both humans and machines to read. This small set of tools allows for simple parsing of all of the data within the PDFs, and allows for the data to be pushed to a database for querying.
Technologies Used:
Python, MongoDB, BeautifulSoup, unpdfer,
Project Lead:
Tim Duffy
